# AI4Aging Workshop Website

Static GitHub Pages site for a proposed NeurIPS 2026 workshop: **Foundation Models and Generative AI for Aging Science**.

**Live URL:** https://ai4aging.github.io

## Edit content

- Public site: `index.html`
- Proposal drafts: `PROPOSAL_NOTES.md` (organizers only)
- Styling: `assets/styles.css`

## Repositories

| Remote | Repository | URL |
|--------|------------|-----|
| `pages` | [ai4aging/ai4aging.github.io](https://github.com/ai4aging/ai4aging.github.io) | https://ai4aging.github.io |
| `origin` | [akramb8i/ai-aging-workshop](https://github.com/akramb8i/ai-aging-workshop) | https://akramb8i.github.io/ai-aging-workshop/ |

## Publish changes

After editing locally, push to both remotes:

```bash
git push pages main    # live site (ai4aging.github.io)
git push origin main   # personal backup repo
```

GitHub Pages is configured on **ai4aging/ai4aging.github.io**: Settings → Pages → deploy from `main` / root.
